Detailed Description

This class solves following equation. $$ | f(x) |^{2} = 0 $$.

Levenberg Marquardt method updates $x$ by the following equation each iteration. $$ x = x - (J_{f}^{T}(x)J_{f}(x) + {diag}(J_{f}^{T}(x)J_{f}(x)))^{-1}J_{f}^{T}(x) f(x) $$
